## Current Implementation:
The FloatingAssistant component uses the custom SVG icon at `/ai_button.svg` and will:
- Display the custom icon in the floating action button (32x32px)
- Display the custom icon in the dialog header avatar (24x24px) 
- Display the custom icon in message avatars (20x20px)
- Fall back to the default Material-UI SmartToy icon if the image fails to load

## AI Icon Design:
- **Format**: SVG with blue gradient and glow effects
- **Theme**: Neural network brain with data processing elements
- **Features**: 
  - Animated gradient from light to dark blue
  - Neural nodes and connections
  - Signal antenna lines

  - Modern, tech-focused aesthetic
- **Optimized**: Works well on both light and dark backgrounds
- **Scalable**: Vector format ensures crisp rendering at all sizes

## Customization:
To use your own icon, simply replace `ai_button.svg` with your custom image (PNG or SVG format supported).
